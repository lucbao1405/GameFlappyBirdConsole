Dự án bài tập lớn môn Lập trình C# - ICTU. Game Flappy Bird được tái hiện trên môi trường Console với tư duy lập trình hướng đối tượng (OOP).

Cấu trúc Project
Entity.cs: Lớp trừu tượng định nghĩa các đối tượng game.

Bird.cs: Quản lý logic chim (di chuyển, nhảy).

Pipe.cs: Định nghĩa hình dáng và tọa độ cột.

PipeManager.cs: Quản lý danh sách cột, va chạm và tính điểm.

GameManager.cs:Điều khiển vòng lặp game và trạng thái màn hình.

Program.cs: Điểm bắt đầu (Entry point) của ứng dụng.

Interface.cs Lưu các interface
